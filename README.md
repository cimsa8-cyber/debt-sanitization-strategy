# Proyecto Finanzas AVN Claude Code

Proyecto de gestión financiera integral para AlvaroVelasco.Net SRL, desarrollado con asistencia de Claude Code.

## Objetivos

- Configuración y actualización de QuickBooks Desktop 2013 con multimoneda (USD/CRC)
- Registro y conciliación de transacciones bancarias y tarjetas de crédito
- Separación clara entre finanzas empresa y personales (sanitización)
- Análisis de deuda y optimización de pagos
- Estrategias de reducción de deuda y mejora de liquidez
- Control de cuentas por cobrar y por pagar
